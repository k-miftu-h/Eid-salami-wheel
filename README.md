# Eid Salami Wheel 🌙✨

A premium, interactive web application for celebrating Eid with a fun "Salami Wheel".

## 🚀 Features
- **10 Reward Tiers**: Custom rewards with Bengali descriptions.
- **Premium UI**: Smooth animations, mobile-first design, and interactive modals.
- **Firebase Powered**: Real-time spin tracking and user validation.
- **Responsive**: Mathematically precise geometry for any screen.

## 🛠 Setup & Tech
- **Frontend**: HTML5, Vanilla CSS3, JavaScript (ES6+).
- **Backend**: Firebase Firestore.
- **Tools**: Modular Firebase SDK (v12.10.0).

## 💡 How to Use
1. Enter your unique **EID-XXXX** status ID.
2. Spin the wheel to win "Salami".
3. View your result in the celebration modal.
